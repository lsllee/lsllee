音效文件说明

1. 基础音效
- correct.mp3：答对音效
- wrong.mp3：答错音效
- pop.mp3：气泡爆破音效
- explosion.mp3：爆炸特效音效

2. 奖励音效
- nice.mp3：连续答对3个的奖励音效
- great.mp3：连续答对7个的奖励音效
- brilliant.mp3：连续答对11个的奖励音效
- excellent.mp3：连续答对16个的奖励音效
- perfect.mp3：连续答对20个的奖励音效

下载音效方法：
1. 访问 https://pixabay.com/sound-effects/ 或 https://freesound.org/ 
2. 搜索相关关键词如"correct"、"award"、"achievement"等
3. 下载免费音效并重命名为上述文件名
4. 放入sounds文件夹中

注意：如未下载音效，程序会自动使用内置的音效生成功能
